# 代码备份管家 v1.0.0 (Windows 98 Edition)

一个采用 Windows 98 / Y2K 风格设计的本地代码备份管理器。扫描并同步本地 Git 仓库代码至您的云盘备份目的地（如 Google Drive 等）。

---

### 核心功能
* **界面设计**：还原 Windows 98 窗口、状态栏和菜单栏，内置深色/浅色/系统同步主题。
* **本地与云端同步**：支持配置本地主目录，并扫描子级项目；支持指定云盘同步目标路径。
* **Git 状态感知**：自动识别 Git 仓库、检查未提交的修改，仅同步 Git 跟踪的文件（避免同步 `node_modules` 等文件）。
* **菜单与排序**：支持按照项目名称、路径、Git 状态等多维度排序与关键字过滤；自带“文件”菜单导入/导出 JSON 配置文件。

---

### 快速启动

#### 1. 前置环境
* 安装 Python 3.x
* 安装 Git 并配置到环境变量中

#### 2. 启动服务
在项目根目录下，运行以下命令启动服务：
```bash
python main.py
```
启动后，系统将在浏览器中打开：[http://localhost:16825](http://localhost:16825)

---

### 项目结构
* `main.py`：Python HTTP 后端，负责本地目录扫描、获取 Git 状态、执行文件同步与配置读取。
* `index.html`：前端交互界面。
* `styles.css`：Windows 98 界面样式表，适配自适应布局与深/浅主题。
* `app.js`：前端控制器（配置加载、数据过滤与排序、同步指令）。
